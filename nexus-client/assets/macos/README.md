# macOS Assets

This directory contains macOS-specific application assets for Nexus BBS.

## Files

- `nexus.png` - 1024×1024px PNG icon
- `generate_assets.sh` - Script to generate PNG from SVG source

## Generating Assets

```bash
./generate_assets.sh
```

This will generate the PNG icon from the SVG source.

**Note:** cargo-bundle automatically converts the PNG to ICNS format when building the macOS app bundle.

## Icon Format

- **PNG**: 1024×1024px with transparency
- **ICNS**: Automatically generated by cargo-bundle from the PNG

## Requirements

- ImageMagick (`magick` or `convert` command)

**Install on macOS:**
```bash
brew install imagemagick
```

**Install on Linux:**
- Arch: `pacman -S imagemagick`
- Debian/Ubuntu: `apt install imagemagick`
- Fedora: `dnf install imagemagick`

## Building App Bundle

```bash
# Install cargo-bundle (one-time)
cargo install cargo-bundle

# Build from project root
cargo bundle --release

# Result: target/release/bundle/osx/Nexus BBS.app
```
